O que é um Sistema Operacional?

O Sistema Operacional (SO) é o software principal que gerencia todo o hardware do computador e serve de ponte entre você (usuário) e a máquina.

EXPLICAÇÃO RÁPIDA

Existem 4 tipos principais de Sistemas Operacionais, cada um projetado para um propósito diferente:

💻

Desktop

Ex: Microsoft Windows, macOS, Linux

  • 👉 Usado em computadores pessoais (PC e notebook)
  • 👉 Foco: usuário final (facilidade, programas variados, jogos)
📱

Mobile

Ex: Android, iOS

  • 👉 Usado em smartphones e tablets
  • 👉 Foco: aplicativos + mobilidade + economia de bateria
🌐

Servidores

Ex: Linux Server, Windows Server

  • 👉 Computadores que “servem” outros dispositivos
  • 👉 Hospedam sites, bancos de dados e serviços (ex: GitHub Pages)
🤖

Embarcados

Ex: FreeRTOS, Linux Embarcado

  • 👉 Estão “dentro” de outros dispositivos
  • 👉 Ex: TV, carro, micro-ondas, geladeira, relógio inteligente
