Os 4 Tipos de Sistemas Operacionais

💻

Desktop

Ex: Microsoft Windows

  • 👉 Usado em computadores pessoais
  • 👉 Foco: usuário final
📱

Mobile

Ex: Android

  • 👉 Smartphones
  • 👉 Foco: apps + mobilidade
🌐

Servidores

Ex: Linux Server

  • 👉 Computadores que “servem” outros
  • 👉 Hospedam sites (inclusive GitHub Pages)
🤖

Embarcados

Ex: sistemas em dispositivos

  • 👉 Estão dentro de dispositivos
  • 👉 Ex: TV, carro, micro-ondas
